Snake Game AI

Beating Snake!

I programmed an algorithm to beat the game of snake. This algorithm was created in two-fold; it implemented the Hamiltonian cycle algorithm and a backtracking algorithm. At the start of the game, it would use a simple backtracking algorithm to predict future game state since finding the Hamiltonian cycle with many possible moves is very computationally expensive. However, when most squares are covered, we switch to the Hamiltonian cycle algorithm to guarantee the win.
